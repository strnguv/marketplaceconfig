To use Filesystem APIs, include a reference in the XAP-producing-project to /Lib/NativeIO.dll
To use Socket APIs, include a reference in the XAP-producing-project to /Lib/NativeIO.dll
For a Windows Phone OS version 7.1 or greater app, you must use NativeIO_Mango.dll instead.

To elevate the process using the HtcRoot project, include a reference to HtcRoot.dll.
